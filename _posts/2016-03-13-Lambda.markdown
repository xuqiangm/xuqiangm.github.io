本文主要介绍的是 Lambda 表达式的简单用法。

# 基本语法是：
- \[上下文变量说明\](Lambda表达式参数表) -> 返回类型 { 语句块 }
- \[上下文变量说明\](Lambda表达式参数表) { 语句块 }

# 代码实例

```
float f0 = 1.0;
std::cout << [=](float f) mutable { return f0 += std::abs(f); } (-3.5);
std::cout << '\n' << f0 << '\n';
```

# lambda引入符
- \[\]    //不捕获任何外部变量
- \[=\]   //以值得形式捕获所有外部变量
